# archive-goto-pg

## 项目简介

`archive-goto-pg` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的存储与展示空间，方便用户浏览、分享或引用已归档的页面内容。

## 目录说明

```
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的公共资源（如样式、脚本、图片等）
├── index.html      # 仓库首页，提供导航或说明
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或文件代表一个独立的归档页面，命名以日期或主题为参考，便于识别。
- **assets/**：若页面需要共用资源（如 CSS、JavaScript、字体等），可统一放置于此。
- **index.html**：仓库的入口页面，可列出所有归档页面链接或提供简要说明。

## 页面归档说明

- 归档页面均为独立的 HTML 文件，可直接通过浏览器打开或部署到静态托管服务。
- 页面内容保持原始形式，不进行修改或适配，仅做存档用途。
- 归档时间以文件创建或提交时间为准，不另行标注。
- 若页面依赖外部资源（如图片、样式表），已尽量内联或随仓库提供，但无法保证所有外部链接长期有效。

## 维护说明

- 本仓库由维护者定期或不定期更新，新增或移除归档页面。
- 欢迎通过 Issue 提出归档建议，但请勿提交与具体网站、推广或营销相关的内容。
- 对于已归档的页面，原则上不进行修改；如需更新，将另行创建新文件。
- 仓库内容仅供存档与参考，不构成任何形式的推荐或认可。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `index.html` 或 `pages/` 下的任一 HTML 文件。
3. 也可将整个仓库部署到任何静态托管服务（如 GitHub Pages、Netlify 等），以便在线访问。

## 许可

本仓库内容仅用于存档与展示，不附带任何特定许可。如有特殊署名或版权要求，请通过 Issue 联系。
